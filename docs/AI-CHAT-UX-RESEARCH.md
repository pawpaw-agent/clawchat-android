## 2. 各项目亮点交互设计

### 2.1 LibreChat — 企业级功能最全

#### 会话列表交互
```
┌─────────────────────────────────────┐
│ 🔍 搜索会话...                      │
├─────────────────────────────────────┤
│ 📁 今日                             │
│   💬 API 调试助手        2:34 PM   │
│   💬 代码审查            1:15 PM   │
│ 📁 昨日                             │
│   💬 部署日志分析        昨天      │
├─────────────────────────────────────┤
│ [+ 新建会话] [📁 归档]             │
└─────────────────────────────────────┘
```

**亮点**：
- ✅ **时间分组**：按今日/昨日/本周自动分组
- ✅ **全局搜索**：搜索所有消息内容
- ✅ **Fork 分支**：从任意消息分叉对话
- ✅ **Presets 预设**：保存/分享预设配置

#### 消息卡片交互
```
┌─────────────────────────────────────────────────┐
│ 🤖 Assistant                                     │
│ ─────────────────────────────────────────────── │
│ 这是回复内容...                                  │
│                                                  │
│ ┌─ 🛠️ Tool: read ────────────────────────┐     │
│ │ 📂 /var/log/error.log                   │     │
│ │ ✅ 完成 · 0.3s                    [展开] │     │
│ └─────────────────────────────────────────┘     │
│                                                  │
│ ─────────────────────────────────────────────── │
│ [📋复制] [🔄重新生成] [✏️编辑] [↩️继续] [🔀Fork] │
└─────────────────────────────────────────────────┘
```

**亮点**：
- ✅ **工具卡片折叠**：工具调用默认折叠，点击展开
- ✅ **Fork 按钮**：直接从消息分叉
- ✅ **继续生成**：让 AI 继续输出
- ✅ **编辑消息**：编辑后重新发送

**亮点**：
- ✅ **Prompt 模板 (Mask)**：预设提示词模板
- ✅ **自动压缩历史**：长对话自动压缩节省 token
- ✅ **分享为图片**：生成精美分享图
- ✅ **Artifacts 窗口**：代码预览独立窗口

#### 输入区交互
```
┌─────────────────────────────────────────────────┐
│ ┌─────────────────────────────────────────────┐ │
│ │ 输入消息...                                 │ │
│ │                                             │ │
│ └─────────────────────────────────────────────┘ │
│ [📎] [🎤] [📝模板]                   [⬆️ 发送] │
└─────────────────────────────────────────────────┘
```

**亮点**：
- ✅ **模板快捷入口**：输入框旁直接选模板
- ✅ **语音输入**：支持语音转文字
- ✅ **快捷键发送**：Enter 发送，Shift+Enter 换行

## 7. 设计原则总结

### 从研究项目中提炼的设计原则

| 原则 | 说明 | 参考 |
|------|------|------|
| **渐进披露** | 复杂功能默认隐藏，按需展开 | LibreChat 工具卡片 |
| **即时反馈** | 操作立即有视觉反馈 | All 发送状态 |
| **上下文操作** | 操作按钮在相关内容附近 | LobeChat 悬停显示 |
| **快捷键优先** | 常用操作支持快捷键 | All Enter/Shift+Enter |
| **可恢复性** | 支持撤销/重做/分支 | LibreChat Fork |
| **信息分层** | 重要信息优先展示 | 时间分组 |
